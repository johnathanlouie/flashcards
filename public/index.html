To start using this program, at least one chapter from any book must be selected. To select a chapter, click on the Library button, click on any book title, then click on a chapter. Any number of chapters, from any number of books can be selected.

Once a chapter has been selected, click the Back button, which looks like the left arrow, to get back to the main screen.

The Previous and Next buttons cycle through the flashcards. The Remove button removes the card that is currently displayed. This is useful if one does not want to continue seeing it. The Reveal button shows the entire card. These 4 buttons only work if at least one chapter is selected.

The Settings button brings up options that control the type of hints on the front side of the card.
